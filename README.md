#saferm
======
saferm is shell script what safty remove command instead to '/bin/rm'.

### install
	cd saferm
	chmod +x saferm
	./saferm -install
	
### usage
	use the 'rm' comman as usual to move to ~/.Trush;-) 
	use 'rm -force [someting]' to remove like '/bin/rm' command.
	use 'rm -clean' to clear the trush.
